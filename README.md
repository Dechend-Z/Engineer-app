# เครื่องคิดเลขอเนกประสงค์ (Utility Calculator)

แอปพลิเคชันเว็บนี้ให้เครื่องมือที่มีประโยชน์ 4 อย่างที่จัดเรียงในรูปแบบสองคอลัมน์:
- เครื่องคิดเลขช่วงเวลาของวันที่
- เครื่องแปลงเลขฐาน (เลขฐานสอง/เลขฐานสิบ/เลขฐานสิบหก)
- เครื่องคิดเลขอายุการใช้งานแบตเตอรี่
- เครื่องมือการนับตัวอักษร

## คุณสมบัติ

### เครื่องคิดเลขช่วงเวลาของวันที่
- คำนวณช่วงเวลาของวันที่ (ในรูปแบบ วัน สัปดาห์ เดือน ปี)
- เพิ่มจำนวนวันในวันที่เฉพาะ
- แสดงรายละเอียดการแบ่งช่วงเวลา

### เครื่องแปลงเลขฐาน
- แปลงเลขฐานสิบ เลขฐานสอง และเลขฐานสิบหก
- การแปลงแบบเรียลไทม์เมื่อคุณพิมพ์
- การตรวจสอบความถูกต้องของข้อมูลนำเข้าสำหรับแต่ละระบบเลข
- ฟังก์ชันล้างเพื่อรีเซ็ตช่องทั้งหมด

### เครื่องคิดเลขอายุการใช้งานแบตเตอรี่
- คำนวณอายุการใช้งานแบตเตอรี่จากความจุ (mAh) และการใช้พลังงานของอุปกรณ์ (mA)
- คำนวณเวลาคายประจุสำหรับเปอร์เซ็นต์การใช้งานเฉพาะ
- คำนึงถึงปัจจัยประสิทธิภาพ
- ให้ผลลัพธ์ในรูปแบบ ชั่วโมง วัน และสัปดาห์

### เครื่องมือการนับตัวอักษร
- นับตัวอักษร (ทั้งที่มีช่องว่างและไม่มี)
- นับคำ ประโยค และย่อหน้า
- คำนวณความยาวเฉลี่ยของคำ

## วิธีการใช้งาน

1. บันทึกไฟล์ทั้งหมด (index.html, styles.css, script.js) ในไดเรกทอรีเดียวกัน
2. เปิด index.html โดยตรงในเบราว์เซอร์สมัยใหม่ใดๆ หรือ
3. สำหรับการทำงานที่ดีขึ้น ให้ใช้เซิร์ฟเวอร์ท้องถิ่น:

```bash
cd /path/to/utility-calculator
python3 -m http.server 8000
```

จากนั้นเยี่ยมชม `http://localhost:8000` ในเบราว์เซอร์ของคุณ

## ความต้องการ

- เบราว์เซอร์สมัยใหม่ที่เปิดใช้งาน JavaScript
- ไม่มีการพึ่งพาภายนอก - การคำนวณทั้งหมดทำงานในเครื่องบนเบราว์เซอร์ของคุณ

## การใช้งาน

1. ค้นหาเครื่องมือที่คุณต้องการบนหน้าเว็บ (จัดเรียงในสองคอลัมน์เพื่อให้เข้าถึงได้ง่ายขึ้น)
2. ป้อนพารามิเตอร์ที่จำเป็นในช่องข้อมูลนำเข้า
3. คลิก "คำนวณ" หรือ "วิเคราะห์" เพื่อดูผลลัพธ์
4. ผลลัพธ์จะปรากฏในพื้นที่ผลลัพธ์ที่กำหนดไว้พร้อมข้อมูลโดยละเอียด

## รายละเอียดทางเทคนิค

- การใช้งานด้วย HTML, CSS และ JavaScript อย่างบริสุทธิ์
- รูปแบบสองคอลัมน์ที่ตอบสนองได้ซึ่งใช้งานได้กับอุปกรณ์เดสก์ท็อปและมือถือ
- การคำนวณทั้งหมดดำเนินการที่ฝั่งไคลเอนต์เพื่อความเป็นส่วนตัวและความเร็ว
- ไม่มีการส่งข้อมูลไปยังเซิร์ฟเวอร์ภายนอก
